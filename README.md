# 🏥 CityCare Hospital Management System

## 📌 Overview

A full-stack hospital management system built using Flask and MySQL. It helps manage patients, doctors, appointments, and medical records efficiently.

---

## 🚀 Features

### 👩‍⚕️ Patient

* Register/Login
* Book appointments
* Cancel/Reschedule appointments
* View history
* Upload reports

### 👨‍⚕️ Doctor

* View appointments
* Manage patient records
* Dashboard analytics

### 🧑‍💼 Receptionist

* Manage appointments
* Approve requests
* Manage doctor schedules

---

## 🗄️ Database

* MySQL database
* Tables:

  * users
  * doctors
  * appointments
  * medical_records
  * notifications
  * doctor_requests

---

## ⚙️ Tech Stack

* Frontend: HTML, CSS, JS
* Backend: Flask
* Database: MySQL

---

## ▶️ How to Run

1. Clone repo
2. Run schema.sql in MySQL
3. Install dependencies:
   pip install -r requirements.txt
4. Run backend:
   python app.py
5. Open index.html
